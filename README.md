# 📚 SQL LeetCode Solutions

Repository ini berisi kumpulan solusi SQL yang saya kerjakan dari platform LeetCode sebagai bagian dari proses belajar dan pengembangan kemampuan analisis data.

Tujuan repository ini adalah untuk mendokumentasikan perjalanan belajar SQL, memperdalam pemahaman konsep database, serta mempersiapkan diri untuk kebutuhan Data Analyst, Data Scientist, dan Business Intelligence.

## 🎯 Topik yang Dipelajari

* SELECT dan Filtering Data
* ORDER BY
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY dan HAVING
* JOIN (INNER, LEFT, RIGHT)
* Self Join
* Subquery
* Common Table Expression (CTE)
* Window Functions
* CASE WHEN
* Date Functions
* Query Optimization
